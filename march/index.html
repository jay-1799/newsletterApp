Product Highlight

ISD Design Project

Introduction:
The ISD Process App is a specialized tool designed to support Instructional System Designers (ISDs) by consolidating and streamlining the course design documentation process. Currently, instructional designers at KeelWorks and other organizations are required to use multiple, disconnected documents to manage course design processes.

These documents include redundant information fields (e.g., course title, project team), and updates in one document often necessitate manual changes in others, leading to inefficiencies, inconsistencies, and potential errors. Moreover, the current method of documenting course design is fragmented. Designers must navigate between different files to access or update various components of the course, such as project details, design specifications, and revisions. This creates a time-consuming process that is prone to duplication of effort and confusion.

Objective:
The ISD Process App will simplify this workflow by bringing all necessary documentation into one integrated platform. The application will serve as a central repository where instructional designers can manage all project and design information in a single interface, reducing the need for repetitive data entry and minimizing the risk of errors in document synchronization.

The app will generate a course blueprint that can be further developed by technical developers and integrated into a Learning Management System (LMS). How Application Helps KeelWorks The system supports the foundation’s design of core competency courses central to the KeelMaster program improving design efficiency. Additionally, monetizing this tool can play a significant role in funding foundation operations as well as the endowment fund essential to supporting change for the economically disadvantaged. The Software as a Service (SaaS) application can be leased to other organizations for improved instructional design results. The application would also create opportunities for future interns to be involved in customization and product maintenance.

Team Highlight

M&M Team

The Google Workspace (GWS) Team at KeelWorks is responsible for managing and optimizing the company’s workspace ecosystem, ensuring security, automation, and efficiency. Our primary focus areas include:

  • User Management & Onboarding: Streamlining access control, onboarding new users, and automating inactive user removal.
  • Security Enhancements: Implementing 2-Step Verification (2SV) enforcement, phishing email mitigation, and permission controls for sensitive data.
  • Automation & Monitoring: Developing scripts for user activity monitoring, GitHub administration, and automating reporting for internal operations.
  • Collaboration & Documentation: Enhancing knowledge-sharing through documentation, setting up Google Drive permissions, and ensuring compliance with best practices.
